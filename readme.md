# Kahve_Siparis_Ekrani

## Proje Tanıtımı 

*Bu projede **.Net** kullanarak müşteriye kahve siparişi vermeye yarayan bir masaüstü uygulaması oluşturdum. CRUD operasyonları için ado.net disconnect mimari kullandım.*
